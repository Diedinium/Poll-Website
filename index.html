What is this webpage about?


This website is intended to be an information resource for younger voters, voting rates among young voters is the lowest in all the demographics in the UK. This is not an official government page, it is an independent project and is supposed to stand on it's own merit. The name of the website is part of this message, we have named the website "Poll" because that's the intention of the site; to get people to the Polls. Political information on this site will be presented in an unbiased and neutral view, so that you can make your own conclusion about the information provided.


The aim of this website is to change that, through information about how our democratic system works, why voting is important and importantly; how to vote. Continue down the page to learn where to head next, or start exploring from the navigation options at the top.
